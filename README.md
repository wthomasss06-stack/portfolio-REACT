## ✨ Caractéristiques

### 🎨 Design & Interface

- **Architecture React** : Single Page Application (SPA) avec composants réutilisables
- **Design moderne** avec effets glassmorphism et animations fluides
- **2 modes d'affichage** :
  - Mode Sombre (par défaut) — vert néon sur fond noir profond
  - Mode Clair — rouge-orange bold sur fond blanc éditorial
- **Responsive design complet** : mobile, tablette, desktop — avec composants mobiles dédiés (carousel, nav verticale)
- **Logo SVG animé** `AkafolioLogo` intégré dans `App.jsx` :
  - Dessin séquentiel du **"A"** trait par trait + point néon spring + pulse infini
  - Texte **"KA"** et **"folio"** animés avec stagger · ligne déco tracée
  - S'adapte aux 2 modes · présent dans loader, navbar et footer
- **Navigation fluide** : Scroll smooth vers les sections
- **Navbar double** : horizontale sur desktop + verticale compacte sur mobile (Material Symbols)

### 🌟 Système de particules

- Mouvement organique avec dérive naturelle, friction légère et drift automatique
- Répulsion au curseur dans une zone de 100px avec force progressive
- Connexions dynamiques entre particules voisines (seuil 130px) avec opacité dégradée
- Halo radial subtil autour de chaque particule (glow canvas)
- Couleurs adaptées au thème : vert néon (sombre) / orange chaud (clair)
